# GatoOscuro
La vida de GatoOscuro; todo tiene una razón de ser.
Se supone que este blog está enfocado tanto en mi vida personal como en el lado más técnico, aunque en ocasiones suelo variar e incluir de todo un poco y como el lema resa "todo tiene una razón de ser" si lo escribo es por algo y tiene un sentido para mi, quizás no tanto para ti.
Este blog no sólo está enfocado a la tecnología como se suele ver en otras blogosfera, aunque se tenga gran cantidad de contenido de dicho tipo, pero en lo absoluto no es solo lo que tengo para ofrencer, va más allá: una vivencia, una experiencia, un estilo de vida, quizás y solo quizás, una trágicomedia particual (eso es cuestión de percepción). 
Agradezco a quienes lean, colaboren y aporten frente a la discusión del contenido, pero todo desde el respeto, que es lo poco que quiero para lograr una efectuosa interacción y más, exponiendo cada rincón de mi alma ¡no es fácil! Gracias.
