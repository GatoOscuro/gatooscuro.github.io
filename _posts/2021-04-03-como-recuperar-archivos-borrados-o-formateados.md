# Cómo recuperar archivos borrados o formateados

Alguna vez en nuestra vida hemos eliminado; archivos, fotos, videos, sin querer de nuestros: ordenadores, celulares y etc. Aquí es donde nos preguntamos **¿Ahora qué hago?**

Tenemos que saber que al borrar un archivo lo que realmente se hace es “marcarlo como espacio reutilizable”, por lo que es fácil recuperarlo (siempre y cuando, no se haya vuelto a escribir encima). Más información acerca de esto y la eliminación de datos privados de la mano de un experto: https://www.emezeta.com/articulos/formatear-un-disco-duro-con-seguridad

A continuación mencionaré los programas gratuitos para recuperar todo tipo de archivos borrados (recuerda ejecutar como administrador los programas utilizados) - **actualización 2021** este artículo fue escrito más enfocado a usuarios Windows, cuando ello utilizaba Windows, aunque también hay herramientas generales que se puede aplicar al resto de S.O sin problemas como lo es R-Linux.

Agrego: es una breve explicación técnica de lo que hace cada aplicación, en ningún momento le doy mi voto de confianza al 100%, con el tiempo las herramientas y sus organizaciones que hay detrás cambian sus políticas de juego y con ello se ve afectada la privacidad del usuario, por ello antes de instalar, verificar la imagen actual.

 
**R-Linux**

Esta herramienta gratuita permite recuperar archivos eliminados, formateados, borrados o dañados tanto en particiones Linux EXT2, EXT3 o EXT4 como en particiones FAT32 o NTFS. Contiene filtros para recuperar información dependiendo del contenido y diferentes tipos de herramientas para usuario avanzado.

Enlace de descarga: https://www.r-studio.com/free-linux-recovery/Download.shtml

![Cosmic radiation](https://windows-cdn.softpedia.com/screenshots/RLinux-Data-Recovery-Utility_1.png)

 

**NTFS Undelete**

Es una buena herramienta para recuperar archivos borrados, pero es orientado a solo discos con particiones en formato NTFS de Microsoft.

Enlace de descarga: https://www.ntfsundelete.com/download

![Cosmic radiation](https://www.indir.org/upload/ntfs_undelete_1.png)


 
**MiniTool Power Data Recovery**

Permite recuperar ficheros eliminados, particiones dañadas, particiones con perdida de datos o unidades USB de cámaras digitales, memorias auxiliares, pendrives o medios ópticos (CD o DVD).

Enlace de descarga para Windows: https://www.minitool.com/data-recovery-software/free-for-windows.html

Enlace de descarga para Mac: https://www.minitool.com/data-recovery-software/free-for-mac.html

**Data Recovery**

Programa gratuito que permite la recuperación de datos de particiones FAT y NTFS, ficheros encriptados EFS.

Enlace de descarga: https://www.easeus.com/data-recovery-software/

![Cosmic radiation](https://www.softwarehow.com/wp-content/uploads/wondershare-data-recovery-software-box.jpg)
 

**Free File Recovery**

Free File Recovery es una utilidad gratuita de Windows para restaurar archivos que se han eliminado accidentalmente del equipo. Esto incluye archivos vaciados de la papelera de reciclaje, así como imágenes y otros archivos que han sido borrados por error del usuario de las tarjetas de memoria de la cámara digital o reproductores MP3. Incluso traerá de vuelta los archivos que han sido borrados de su iPod, o por errores, caídas y virus!

Enlace de descarga: https://www.avnmedia.com/free_file_recovery.html

![Cosmic radiation](https://www.avnmedia.com/screenshot/free_file_recovery.gif)

 

**Pandora Recovery**

De la misma manera que los datos pueden perderse, pueden recuperarse intactos en la mayoría de los casos con la solución de recuperación de datos adecuada. El taladro de disco gratuito es la nueva generación de recuperación de datos de Pandora y hereda su funcionalidad de recuperación de archivos más destacada y la completa con un sinfín de algoritmos modernos que aumentan la eficacia de la recuperación de datos, proporcionan resultados más concisos y una experiencia de usuario más agradable.…

Enlace de descarga: https://www.pandorarecovery.com/

![Cosmic radiation](https://downloadbasket.com/wp-content/uploads/2015/09/Pandora-Recovery_1.png)

 

**Undelete Plus**

Aplicación freeware para recuperar archivos, disponible en versión instalable y portable. La interfaz recuerda mucho al nuevo estilo de aplicaciones de Microsoft y tiene varios tipos de filtro de búsqueda.

Enlace de descarga: https://undelete360.com/download.html

![Cosmic radiation](https://www.samomoi.com/logiciels/images/undelete.png)

 

**SoftPerfect File Recovery**

Software de recuperación de archivos gratuito y potente para recuperar 2 GB de archivos de forma gratuita. Resuelve todos los problemas de pérdida de archivos: recuperación de archivos vaciados de la Papelera de reciclaje, perdidos debido a un fallo de software, disco duro formateado o dañado, ataque de virus, partición perdida u otras razones desconocidas bajo Windows 8.1/8/7/Vista/XP/2000 y Windows Server 2012/2008/2003.

Enlace de descarga tanto para Windows y Mac: https://www.softperfect.com/products/filerecovery/

![Cosmic radiation](https://img.creativemark.co.uk/uploads/images/477/10477/largeImg.png)
 

**Recuva**

Programa para recuperar archivos eliminados llamado Recuva. El programa permite realizar una búsqueda rápida o profunda, tanto en discos duros como iPod, emails borrados accidentalmente en aplicaciones como Outlook o Thunderbird, discos dañados o formateados, etc.

Enlace de descarga: https://www.ccleaner.com/recuva

![Cosmic radiation](https://s1.pir.fm/pf/rc/tmprcui.png)

 

**Glary Undelete**

Se trata de una atractiva y sencilla aplicación para recuperar archivos eliminados.

Enlace de descarga: https://www.glarysoft.com/glary-undelete/

![Cosmic radiation](https://img.tamindir.com/ti_e_ul/canerdil/p/glary-undelete_1_770x548.jpg)
